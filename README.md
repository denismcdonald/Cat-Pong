# Cat Pong
An heuristic Pong-clone project, playable via [Glitch](https://cat-pong.glitch.me/). This was my first game and first experiment with the HTML `canvas` element. The project also makes use of [Howler](https://github.com/goldfire/howler.js) for the game's audio.
